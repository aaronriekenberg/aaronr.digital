The UNIX workstation everyone wanted in about 2003?

  • Raspberri Pi OS Lite 64-bit manual install of xorg and other packages.
  • CTWM Window Manger
  • links2 web browser (surprisingly fast and usable on many pages, no javascript).
  • xterm, xclock, xload. Upper xterm is running: 
    top -o RES
    sorting processes by resident memory usage.
  • A usable desktop and webserver in ~275MB of RAM (out of 1GB on the RPI 3B).
